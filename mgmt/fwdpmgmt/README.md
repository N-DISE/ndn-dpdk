# ndn-dpdk/mgmt/fwdpmgmt

This package implements [forwarder data plane](../../app/fwdp/) management.

## DpInfo

**DpInfo.Global** returns global configuration.

**DpInfo.Input** reports about a FwInput.

**DpInfo.Fwd** reports about a FwFwd.

**DpInfo.Pit** reports about the PIT in a FwFwd.

**DpInfo.Pit** reports about the CS in a FwFwd.
